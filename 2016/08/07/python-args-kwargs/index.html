Python的 *args 和 **kwargs

发表于 2016-08-07   |   分类于 Python   |  
分析 *args

当你不确定函数在使用时会有多少个参数传入时，可以使用[*args]定义函数的参数，args类型为 tuple。

分析 **kwargs

与[args]相似，[*kwargs]在你不知道传入参数数量的时候使用，kwargs 类型为 dict。
额外功能是，还可以指定参数的名称。
